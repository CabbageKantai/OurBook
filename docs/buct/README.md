
# 大学综述  

> 本文于2016年录入/高2015级21班唐宏科  

北京化工大学（BeijingUniversityofChemicalTechnology）是中华人民共和国教育部直属的一所以化工为特色的多科性全国重点大学，是国家“211工程”、“985优势学科创新平台”重点建设院校之一，入选“卓越计划”、“111计划”，是北京高科大学联盟的重要成员。

1.大学历史

北京化工大学（原名北京化工学院）始建于1958年，隶属原化学工业部领导，1960年10月中共中央决定把我校列为全国重点大学。建校初期学校设有3个系12个本科专业，在校生规模达3150人，校舍建筑面积9万平方米。

"文革"期间，学校遭到严重破坏，1966年-1970年5年间，学校停课，中断招生。1971年-1976年招收6届工农兵学员。1971年10月与北京化纤工学院合并（校名为北京化工学院），这期间学校对专业进行了大幅度调整，停办了全部尖端绝密专业，增设了若干新专业，全院共设置5个系14个专业。

1977年10月恢复统一高考招生制度，招收四年制本科生，1978年招收首届研究生，同年10月北京化纤工学院从我校分出。

1994年2月学校更名为北京化工大学，1996年4月原化工管理干部学院并入，1998年9月划转为教育部直属高校，2001年7月通过国家"211工程""九五"期间建设项目验收。

4.学术成就

（1）科研机构

截止2014年，学校有2个国家重点实验室、1个国家工程实验室、1个国家工程技术中心、2个发改委批准的基础研究实验室、3个教育部重点实验室、5个教育部工程研究中心、2个化工行业工程技术中心、2个北京市重点实验室、1个北京市工程研究中心、1个北京市哲学社会科学研究基地

1）.国家重点实验室

化工资源有效利用国家重点实验室、有机无机复合材料国家重点实验室

2）.国家级工程技术研究中心

国家碳纤维工程技术研究中心

3）.国家工程实验室

轮胎设计和设计工艺工程实验室

4）.国家发展和改革委员会批准的基础研究实验室

新危险化学品评估及事故鉴定基础研究实验室、危险化学品生产故障预防及监控研究实验室

此外，我校还有11个省、部级重点实验室，15个省、部级工程技术研究中心。

5.杰出校友

1）.贺国强

贺国强，男，1943年生，湖南湘乡人，1966年1月入党，1966年9月参加工作，北京化工学院（今北京化工大学）无机化工系无机物工学专业毕业，大学学历，高级工程师。十七届中央政治局委员、常委，中央纪律检查委员会书记。

2）.程维

程维，小桔科技创始人兼CEO，该公司推出了网络智能叫车系统——滴滴打车，并成为打车软件领域的领导企业。程维曾在阿里巴巴履职八年，期间成为阿里巴巴最年轻的区域经理，后担任支付宝B2C事业部副总经理。2012年，程维从支付宝离职创立了小桔科技。（滴滴打车创始人）

3）.周小川

周小川，男，1948年1月生，江苏宜兴人。1975年毕业于北京化工学院（今北京化工大学），1985年在清华大学获工学博士学位，研究员。1968年7月参加工作，1986年3月加入中国共产党。现任中国人民银行行长，十二届全国政协副主席。

4）.李学勇

李学勇，男，汉族，1950年9月生，河北石家庄人，1974年3月加入中国共产党，1968年6月参加工作，1982年北京化工学院（今北京化工大学）高分子系高分子化工专业毕业，大学学历，工学学士学位，工程师。现任十八届中央委员，省委副书记、省长、省政府党组书记。

6.校友建设

北京化工大学校友总会成立于1988年，在校领导及有关部门、校友及社会各界人士的大力支持和协助下，校友总会工作正在积极稳步地发展。

校友总会的工作宗旨是：继承和弘扬北京化工大学的优良传统；团结和联络海内外广大校友，促进校友之间、校友与母校之间的联系，增进友谊，加强合作；指导各地校友组织工作，开展各类交流活动；支持校友在各自的事业中开拓创新；不负莘莘学子爱校之心、兴校之愿，会同广大校友，推动学校各项事业的发展。

校友总会创办了《校友通讯》期刊，成为传播母校信息，报道校友业绩，联系校友感情，弘扬化大精神的窗口。2008年，校友总会协助学校举办了五十周年校庆活动；2012年7月，由现任校长、工程院院士谭天伟教授担任校友总会会长，合作交流部部长何雨骏担任秘书长。

目前，我校已在各地建立41个校友分会。

7.大学见闻

北京化工大学的学风在我所接触的大学中算是很良好的，现阶段大一、大二都在昌平校区，大三大四会搬至位于东三环的本校区，本校区距离三里屯特别近，但北校区位置偏远，附近有中国政法大学、中国石油大学，再此顺便提醒各位想要报考这两所大学的学弟学妹，中政和中石油都位于昌平这个北六环以外的区。

现在新修建的南口校区一期工程预计会在2017年暑期完工，这意味着从我们这一届开始，大三将迁往南口校区，大四返回东三环的本校区，所以下一届的学弟学妹你们还是会在大一来到昌平，大二大三迁往南口，大四返回本校区，不过再下一届的新生将直接到南口入学。

我知道这可能会让很多向往北京的学弟学妹失望，因为南口是一个比昌平还要遥远的地方，我没有去过那里因为实在是太远了，但是南口校区在竣工后会成为国内基础设施排名前三的校区，也会拥有各种高端的实验室，这是肯定的，我印象中是在我大四毕业的时候南口将全面竣工（虽然这和我没什么关系）。

还有一个不知道是喜是悲的消息，从南口进城可以坐火车，不过具体时间众说纷纭，路途长短从半小时到一个半小时说法不等，可以直接坐到西直门（北京北站）。

目前昌平校区附近的娱乐设施极少，校内新疆西藏东北人很多很多很多，有一个类似万达的商城--金隅万科，还有一些台球厅和极少的酒吧。

还有就是饮食方面，学校有对清真学生设置专门的食堂，但是作为一个土生土长的重庆人，我必须说我真的受不了学校的食堂，给我一瓶老干妈我就可以征服地球！

这边的烧烤基本上都是铁板烧，东北铁板烧，是我现在不习惯、以后也不会习惯的烧烤，他们刷的是酱！每一次都含恨吃下去，然后缅怀我逝去的蓝精灵对面的烧烤摊和南开后门坡上的烧烤。

虽然我画风开始转为吐槽了，但是我必须负责任的告诉你们，北京化工大学的理工类专业非常出色，个别专业在世界上排名前列，高分子材料、材料、化工这三个专业是北化最好的专业，在这些专业领域甚至超过清华北大，如果有学弟学妹是真正想要读这些专业，是想要深入研究的，我是真诚的向你推荐北京化工大学的，虽然现在昌平基础设施也有限，但是北化从来不亏待实验室。

因为大一大二都在偏远的昌平，坐地铁进城往往要用去两个小时，我们和同在昌平的中政法、中石油结下了深刻的革命友谊，我们经常辗转在友校的自习室和图书馆，顺便还可以认识一下兄弟院校的优秀学子。

北化还是一个很适合学习的地方，大家的心思都相对单纯，即使磕磕绊绊，我们到底也会慢慢适应，并且融入其中。

宏德博学，化育天工。这是我们的校训，我认为这也是最适合这个学校的一句话。

8.大学感悟

进入北化接近一年，我加入了很多学生组织，最后选择留在校会。

在这一年中，坦诚地说我真的没有怎么学习，从南开到这里我反而有了心理落差，我为我是一名南开人而骄傲自豪，南开开放自由的氛围、百花齐放的社团、以及各种各样的校级活动真的是比很多大学做得更好。我没有放下过去重新开始，我没有疯狂的学习知识，反而用不同的社会活动来充实自己，然后在考试前临阵磨枪。

北京化工大学是我的第五志愿，我离第四志愿北京林业大学差了一分，当时填报高考志愿和爸爸较劲没有在提前批填报国际关系学院，现在回首也没有多么遗憾。

我在北化遇见了很多很好很好的人，也遇到了现在的恋人，尽管有很多坎坷，但是我想我也找到了自己的梦想，既是要付出非常大的代价，我还是不会放弃追逐。

以前没有目标的日子的确浑浑噩噩，我也是一个很有惰性的人，但是进入大学以来，遇见的人都会鼓励我前行。

从我的角度出发，我从来不觉得学生组织有什么不好，我想在你们进入大学之后会有很多人告诉你们学习最重要，其他的都是虚妄，但是我是一名南开er，我的生活从始至终就是丰富多彩的，南开的六年让我不至于被各种社团和组织迷花了眼，不至于让我每天疲于奔命筋疲力尽，我现在的恋人和最好的朋友都是我在我们部门认识的。我想说的是，学习固然重要，然而在我们的大学里，参加一个组织也是必不可少的精彩部分。

不要死读书，我现在马上就要大二了，我的想法可能也不太成熟，但是我是曾经疯玩过、完全抛下过学习的人，我的大一没有什么很励志的故事，也没有什么努力的过程，我过着和高中截然相反的放纵生活，但那种愉悦始终是短暂的，长久的幸福之余在不断地充实自己中才能得来。

最后我想告诉你们，无论进入哪所大学，多认识一些朋友，你会遇到值得相处一生的人，也不要断开在南开的一切。无论身处怎样的环境，都要保持一份平常的心态，不妄自菲薄、不骄傲自满。

允公允能，日新月异。南开所给你们的，是你们一生中最宝贵的财富。
